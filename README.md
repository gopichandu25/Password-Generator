# Password Generator (Python)

This is a simple and secure **Python Password Generator** project that creates strong, random passwords based on user preferences.

# Features

* Generates strong and random passwords
* Choose password length
* Includes:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* User-friendly and customizable
* Prevents weak passwords
